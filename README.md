# _The NFT Marketplace_

This is an NFT Marketplace, fully compliant with ERC721 Etherium Smart Contract.

### Build With

- `React`
- `Solidity`
- `Genache`
- `Truffle`
- `Web3`
- `Material Design Bootstrap`

## INSTALLATION

fork or download the repository

```npm
npm install
```
